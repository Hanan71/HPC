# 🚀 HPC-Intilaqa - Sustainable Energy Generation Platform

HPC-Intilaqa is an innovative smart platform designed to bridge the gap between physical fitness and environmental sustainability. By leveraging Internet of Things (IoT) technologies, the platform converts kinetic energy and physical activity exerted on gym equipment into clean, sustainable electrical energy, backed by an intelligent rewards and incentive system.

---

## 🌟 Key Features

* **Smart Energy Harvesting:** Converts physical effort and burned calories into actual electrical energy to support the environment.
* **Intelligent Dashboard:** Enables users to track their daily harvested energy, earned points, and vital fitness metrics.
* **Incentive & Rewards System:** Gamifies the experience by converting generated energy into redeemable points and rewards to ensure user retention and engagement.
* **IoT Integration:** Seamlessly connects the platform's software with fitness hardware for real-time, highly accurate data streaming.
* **Responsive Layout:** A modern, clean user interface optimized for a smooth user experience across all screen sizes and mobile devices.

---

## 🛠️ Tech Stack

### Frontend
* **HTML5 & CSS3:** Structured for clean layout and responsive design.
* **Bootstrap 5:** For a flexible, mobile-first responsive grid system.
* **JavaScript (jQuery):** Powers the core interactive elements and dynamic components.
* **Animation & UI Libraries:** Animate.css, Owl Carousel, WOW.js.

### Fonts & Icons
* **Fonts:** Google Web Fonts (Inter, Roboto, Cairo).
* **Icons:** FontAwesome v5, Bootstrap Icons.
